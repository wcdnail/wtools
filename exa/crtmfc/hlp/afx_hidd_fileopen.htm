Диалоговое окно "Открытие файла"

Следующие параметры позволяют указать, какой файл следует открыть:

Имя файла

Указывает файл, который следует открыть. Этот список содержит файлы с расширением, выбранным в поле Тип файлов.

Тип файлов

Указывает тип файла, который следует открыть:

<< Введите список типов файлов для вашего приложения. >>

Диски

Указывает диск, на котором <<приложение>> сохраняет файлы, которые следует открыть.

Каталоги

Указывает каталог, в котором <<приложение>> сохраняет файлы, которые следует открыть.

Сеть

Нажмите эту кнопку, чтобы подключиться к сетевому ресурсу и назначить ему новую букву диска.
