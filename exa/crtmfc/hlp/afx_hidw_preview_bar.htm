Панель предварительного просмотра

Панель предварительного просмотра содержит следующие кнопки:

Печать

Открывает диалоговое окно Печать для запуска задания печати.

Следующая страница

Предварительный просмотр следующей печатной страницы.

Предыдущая страница

Предварительный просмотр предыдущей печатной страницы.

Одна страница / две страницы

Предварительный просмотр одной или двух печатных страниц одновременно.

Крупнее

Увеличивает масштаб печатной страницы.

Мельче

Уменьшает масштаб печатной страницы.

Закрыть

Возвращается из режима предварительного просмотра в окно редактирования.
