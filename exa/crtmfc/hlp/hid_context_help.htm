Команда "Контекстная справка"

Эта команда служит для получения справки по определенной части <<приложения>>. При нажатии кнопки Контекстная справка на панели инструментов, указатель мыши примет вид стрелки со знаком вопроса. Затем щелкните в любом месте окна <<приложения>>, например другую кнопку панели инструментов. Для нажатого элемента будет отображен раздел справки.
