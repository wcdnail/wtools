Команда "Отменить/Нельзя отменить" (меню "Правка")

<< Интерфейс пользователя приложения для команды "Отменить" может отличаться от описанного ниже. Измените соответственно текст справки. >>

Эта команда служит для отмены последнего изменения, если это возможно. Имя команды изменяется в зависимости от того, каким было последнее действие. Команда меню Отменить меняется на команду Нельзя отменить, если последнее действие необратимо.
