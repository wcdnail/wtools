Команда "Закрыть" (меню "Файл")

Эта команда служит для закрытия всех окон с активным документом. В <<приложении>> выдается запрос на сохранение изменений в документе перед его закрытием. Если закрыть документ без сохранения, все изменения, выполненные с момента последнего сохранения будут утеряны. Перед закрытием документа без названия <<приложение>> отобразит диалоговое окно Сохранение документа и предложит ввести имя и сохранить документ.
