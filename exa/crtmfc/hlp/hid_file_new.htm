Команда "Создать" (меню "Файл")

Эта команда служит для создания нового документа в <<приложении>>. Выберите в диалоговом окне Создание файла тип нового файла, который необходимо создать. <<Удалите предыдущее предложение, если приложение поддерживает только один тип документа. >>

Для открытия существующего документа используйте команду "Открыть".
