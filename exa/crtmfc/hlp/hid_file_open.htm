Команда "Открыть" (меню "Файл")

Эта команда служит для открытия существующего документа в новом окне. Одновременно можно открыть несколько документов. Меню Окно служит для переключения между открытыми документами. См. о команде Окно 1, 2, ... .

Для создания новых документов используйте команду "Создать".
