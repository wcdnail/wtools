Команда "Сохранить" (меню "Файл")

Эта команда служит для сохранения активного документа в указанную ранее папку с текущим именем. При первом сохранении документа в <<приложении>> откроется диалоговое окно Сохранение документа для ввода имени документа. Если перед сохранением существующего документа необходимо изменить его имя и папку, где он хранится, выберите команду Сохранить как.
