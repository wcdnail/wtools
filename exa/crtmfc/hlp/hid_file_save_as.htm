Команда "Сохранить как" (меню "Файл")

Эта команда служит для указания имени текущего документа и его сохранения. В <<приложении>> откроется диалоговое окноСохранение документа для ввода имени документа.

Чтобы сохранить документ с текущим именем в папку, где он был сохранен ранее, используйте команду Сохранить.
