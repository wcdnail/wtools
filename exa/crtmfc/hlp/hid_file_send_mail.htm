Команда "Отправить" (меню "Файл")

Эта команда служит для отправки активного документа по электронной почте. Она открывает окно сообщения электронной почты с активным документом во вложении. Можно ввести значения в поля Кому, Тема и т. п., а затем добавить основной текст сообщения. По завершении для отправки сообщения следует нажать кнопку Отправить.
