Команда "Закрыть" (оконное меню)

Используйте эту команду для закрытия активного окна или диалогового окна.

Двойное нажатие кнопки оконного меню равносильно выбору команды Закрыть.

Примечание. Если для одного документа открыто несколько окон, при выборе команды Закрыть в оконном меню документа закрывается только одно окно. Можно также закрыть все окна одновременно с помощью команды Закрыть в меню Файл.
