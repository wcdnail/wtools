Команда "Переключиться" (оконное меню приложения)

Эта команда служит для отображения списка всех открытых приложений. Этот список задач служит для переключения или закрытия приложения в списке.

Элементы диалогового окна

При выборе команды Переключиться появится диалоговое окно со следующими параметрами:

Список задач

Указывает приложение, которое следует закрыть или активизировать.

Переключиться

Делает активным выбранное приложение.

Завершить задачу

Закрывает выбранное приложение.

Отмена

Закрывает список задач.

Каскадом

Упорядочивает открытые приложения, чтобы они перекрывались таким образом, что виден заголовок каждого из них. Этот параметр не применяется к приложениям, свернутым в значки.

Рядом

Упорядочивает открытые приложения в неперекрывающиеся окна. Этот параметр не применяется к приложениям, свернутым в значки.

Упорядочить значки

Упорядочивает значки всех свернутых приложений в нижней части экрана.
