Команда "Строка состояния" (меню "Вид")

Эта команда служит для отображения или скрытия строки состояния, которая описывает действия для выполнения при выборе определенной команды меню или нажатии кнопки на панели инструментов, а также состояния блокировки клавиатуры. При отображении строки состояния рядом с командой в меню появляется флажок.

Для получения справочных сведений о строке состояния см. Строка состояния.
