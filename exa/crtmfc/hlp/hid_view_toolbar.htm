Команда "Панель инструментов" (меню "Вид")

Эта команда служит для отображения или скрытия панели инструментов, содержащей кнопки наиболее общих команд <<приложения>>, например Открыть файл. При отображении панели инструментов рядом с командой в меню появляется флажок.

Для получения справочных сведений о панели инструментов см. Панель инструментов.
