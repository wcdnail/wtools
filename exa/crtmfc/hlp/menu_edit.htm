Команды меню "Правка"

Меню Правка содержит следующие команды:

Отменить Отмена предыдущей операции редактирования.
Вырезать Удаляет данные из документа в буфер обмена.
Копировать Копирует данные из документа в буфер обмена.
Вставить Вставляет данные из буфера обмена в документ.
Вставить связь Вставляет ссылку на данные из другого приложения из буфера обмена.
Вставить объект Вставляет и внедряет в документ объект, например диаграмму или формулу.
Связи Составьте список ссылок на внедренные документы и отредактируйте эти ссылки.
