Команды меню "Файл"

Меню Файл содержит следующие команды:

Создать Создает новый документ.
Открыть Открывает существующий документ.
Закрыть Закрывает открытый документ.
Сохранить Сохраняет открытый документ в файл с тем же именем.
Сохранить как Сохраняет открытый документ в файл с указанным именем.
Печать Печатает документ.
Предварительный просмотр Отображает документ на экране в том виде, в котором он будет напечатан.
Настройка печати Выбирает принтер и подключения к принтеру.
Отправить... Отправляет активный документ по электронной почте.
Выход Закрывает <<приложение>>.
