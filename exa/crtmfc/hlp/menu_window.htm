Команды меню "Окно"

Меню "Окно" содержит команды, которые позволяют размещать в окне приложения несколько документов в различных представлениях:

Новое окно Создает новое окно для просмотра того же документа.
Каскадом Расположение окон на экране с перекрытием.
Рядом Расположение окон на экране без перекрытия.
Упорядочить значки Упорядочивает значки закрытых окон.
Разделить Разделяет активное окно на области.
Окно 1, 2, ... Переходит к указанному окну.
